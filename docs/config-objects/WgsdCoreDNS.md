# WgsdCoreDNS

`WgsdCoreDNS` 由 `network.wgsd-coredns` 插件使用，用来运行一份带有
`wgsd` 插件的 CoreDNS。它主要服务 WireGuard DNS-SD endpoint discovery：
将多个 WireGuard interface 暴露为 `_wireguard._udp.<domain>` 记录。

这是单例配置，名称固定为 `system`。DROS 不内置 `wgsd-coredns` 二进制，`gw bootstrap`
也不会自动安装它。需要时应先运行 `gw install wgsd-coredns`，将 upstream
`wgsd-coredns` 安装为 `/usr/local/bin/coredns`。

当用户提供 `WgsdCoreDNS/system` 配置对象时，DROS 会写入
`/etc/dros/wgsd/Corefile`。如果 `/usr/local/bin/coredns` 已安装，则同时写入
`/etc/systemd/system/dros-wgsd-coredns.service`，并启动/启用
`dros-wgsd-coredns.service`；如果二进制不存在，则只输出 warning 并跳过 service。

## 内置配置

DROS 没有内置 `WgsdCoreDNS` ConfigObject，也没有内置 `wgsd-coredns` 二进制。
`gw install wgsd-coredns` 默认使用固定的 upstream release：

- Release 页面：`https://github.com/jwhited/wgsd/releases/tag/v0.3.6`
- 默认下载：`https://github.com/jwhited/wgsd/releases/download/v0.3.6/wgsd-coredns_0.3.6_linux_amd64.tar.gz`

不想从 GitHub 下载时，也可以提供本地或镜像中的已解压二进制：

```bash
gw install wgsd-coredns /path/to/coredns
gw install wgsd-coredns https://mirror.example/coredns
```

代码默认字段如下：

```yaml
spec:
  bind: 127.0.0.1
  listen: 5304
  interfaces: []
```

## 示例

```yaml
apiVersion: dros/v1alpha1
kind: WgsdCoreDNS
metadata:
  name: system
spec:
  bind: 127.0.0.1
  listen: 5304
  interfaces:
    - name: wg-foo
      domain: foo.wg.natureself.site
    - name: wg-bar
      domain: bar.wg.natureself.site
```

生成的 Corefile 形态：

```caddyfile
# Generated by DROS. Manual changes will be overwritten.
# Resource: WgsdCoreDNS/system

(common) {
    bind 127.0.0.1
    cache
}

.:5304 {
    import common
    wgsd foo.wg.natureself.site. wg-foo
    wgsd bar.wg.natureself.site. wg-bar
}
```

## 字段说明

`metadata.name`

必须为 `system`。

`spec.bind`

CoreDNS 监听地址，默认 `127.0.0.1`。DROS 会写入 `(common)` snippet 中的
`bind` 指令。必须是单行值。

`spec.listen`

CoreDNS 监听端口，默认 `5304`。取值范围为 `1..65535`。

`spec.interfaces`

需要通过 `wgsd` 暴露的 WireGuard interface 列表。启用该配置时必须至少包含一项。
每一项包含：

`name`

WireGuard interface 名称，如 `wg-foo`。必须是单行值，且不能包含空白字符。

`domain`

该 interface 对应的 DNS zone，如 `foo.wg.natureself.site`。DROS 渲染 Corefile 时会
自动补齐末尾的 `.`。
